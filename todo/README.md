# Coinstrat

**[Coinstart](https://www.coinstrat.com/)** components built with [React](https://reactjs.org/). 



##  Install
Install the package in your project directory with:
```
 npm install coinstrat
```

##  Usage

```
 import { Button } from 'coinstrat';
```
